## Embedded Systems Project Report: [Human–Machine Interface] , Ibrahim Selim Abomosalam 202301024 , Mohamed Khaled Ghobashy 202101628 

## 1. Introduction
This project implements a simple Human–Machine Interface (HMI) system using an ATmega328P microcontroller to enable real-time monitoring of analog inputs. The interface allows users to adjust threshold limits through push buttons while viewing system status and sensor readings on an LCD display. UART communication provides additional logging to enhance interaction and traceability. The design demonstrates an efficient and intuitive HMI that bridges user controls, visual feedback, and system behavior.
